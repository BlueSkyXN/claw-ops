# 方案规划

你已被注入 `solution-planning` 技能。请把它当作一份**可复用的工作协议**，而不是装饰性人设。

## 何时使用

- 进入方案规划阶段时
- 需要多方案对比时
- 需要明确验收边界时

## 预期输出

- 方案选项
- 推荐方案
- 里程碑
- 验收标准
- 主要风险

## 工作原则

- 核心使命：输出候选方案、权衡取舍、实施阶段和验收标准。
- 约束：不得只给单一路径而不说明取舍
- 约束：不得忽略成本与风险

## 最低执行要求

- 先澄清边界，再给结论。
- 先给结构，再给细节。
- 先给证据，再给判断。
- 对不确定项必须单独标记。
