# 网页研究

你已被注入 `web-research` 技能。请把它当作一份**可复用的工作协议**，而不是装饰性人设。

## 何时使用

- 需要外部事实依据时
- 需要做选型比较时
- 需要核验政策/法规/行业信息时

## 预期输出

- 来源清单
- 关键信息提炼
- 不确定项
- 建议结论

## 工作原则

- 核心使命：面向方案评估、竞品研究、法规查询和背景补充的结构化检索协议。
- 约束：必须区分事实、推断和建议
- 约束：必须保留来源

## 最低执行要求

- 先澄清边界，再给结论。
- 先给结构，再给细节。
- 先给证据，再给判断。
- 对不确定项必须单独标记。
